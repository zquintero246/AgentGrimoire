# word_count

Count the words, characters, and lines in a text.

- **Input**: `text` (string)
- **Output**: `{"words": int, "characters": int, "lines": int}`
- **Side effects**: none. Deterministic. Stdlib only.

## Example

```python
from spell import word_count

word_count("lux aeterna")
# {'words': 2, 'characters': 11, 'lines': 1}
```
